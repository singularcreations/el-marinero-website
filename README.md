# El Marinero Family Restaurant

Single-page website built with Next.js, TypeScript, Tailwind CSS, and App Router.

## Sections

- Home
- Menu
- Contact

## Public Assets

Place these files in the `public` folder:

```text
public/logo.png
public/slide-1.jpg
public/slide-2.jpg
public/slide-3.jpg
public/menu.pdf
```

The Menu button already points to `/menu.pdf`.
