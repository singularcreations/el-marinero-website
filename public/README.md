# Public Assets

Place these files in this folder:

```text

slide-1.jpg
slide-2.jpg
slide-3.jpg

```

The logo uses `public/logo.png`, the hero slider uses `public/slide-1.jpg`, `public/slide-2.jpg`, and `public/slide-3.jpg`, and the Menu button opens `public/menu.pdf` from the root `index.html`.
